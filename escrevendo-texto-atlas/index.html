Escrevendo Texto com um Atlas

Usa a biblioteca WebGL-fonts para escrever texto na tela usando glifos em um atlas renderizados com WebGL.

Instruções

Escolha a fonte usada para renderizar o texto. A biblioteca WebGL-fonts usa Signed Distance Fields (SDF) para desenhar os glifos de forma eficiente.

Para cada fonte, ela requer um arquivo .png (o atlas de glifos) e um arquivo .json (com as métricas dos glifos). Para transformar um arquivo .ttf (TrueType Font) nos arquivos .png e .json esperados pela biblioteca, você pode usar a ferramenta sdf_atlas.
