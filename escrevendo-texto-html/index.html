Escrevendo Texto com HTML

Usa elementos HTML sobrepostos ao canvas WebGL para escrever texto na tela.

Olá, mundo!

Este é um exemplo de como escrever texto usando elementos HTML sobrepostos ao canvas WebGL.

Você pode usar qualquer tag HTML para formatar o texto, como negrito, itálico, listas, links, etc.

Instruções

Tanto o <canvas> quanto a <div> que contém o texto ficam sobrepostos (div em cima do canvas) e com o mesmo tamanho. Assim, o texto parece estar "dentro" do canvas.

O legal é que dá para usar todo o poder do HTML e CSS para formatar o texto, como mudar fontes, tamanhos, cores, adicionar links, imagens, etc. O ruim é que não é possível que o texto faça parte da cena 3D, ou seja, ele sempre ficará "em cima" do canvas.
